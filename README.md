**eXplainable Mapping Analytics Platform (XMAP)** is a people-centric analytics solution aiming at providing an effortless and systematic methodology to handle complex data analytics tasks. By taking advantages of advanced artificial intelligence (AI) algorithms, XMAP can cope with big and high-dimensional data without compromising the interpretability of the obtained models.
XMAP steps include:
* Data Cleaning
* Data Mapping
* Topology Learning
* Context Learning

XMAP is an ongoing project and purely developed using Python.
